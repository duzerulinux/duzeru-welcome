 Codigo - Code

Este momento é muito importante no desenvolvimento DuZeru. O projeto esta sempre no processo de produção de software livre para o DuZeru. A nossa filosofia é código básico, design simples e dependência mínimas no PC. Nossas aplicações devem ser limpas, rápido, simples e inteligente para o projeto, sempre com grande atenção aos detalhes. Usamos o GPL v3 e se você e um desenvolvedor e gostaria de ajudar na DuZeru com código, dicas, scripts, sugerir ou outra, entre em contato para e-mail claudiosilva@duzeru.org ou acesse o nosso github

This is a very exciting time in DuZeru development. We are in the process of producing custom, free software for DuZeru. Our philosophy is basic code, simple design and minimal dependency. Our applications should be clean, fast and simple. Intelligent, well thought out design with great attention to detail. We use the GPL v3. If you're a developer and would like to help on DuZeru or code, tips, suggest or other, enter contact for e-mail claudiosilva@duzeru.org. Visit our gihub
